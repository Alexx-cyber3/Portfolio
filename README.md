# Arish M — Portfolio

Cybersecurity analyst portfolio built with TanStack Start + Vite + Tailwind v4.

## Setup
```bash
bun install    # or: npm install
bun run dev    # or: npm run dev
```
Site runs at http://localhost:8080

## Build
```bash
npm run build
```
